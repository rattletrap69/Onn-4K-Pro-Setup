Onn 4K Pro Master Blueprint

The Step-by-Step Guide for David & Chris's Low-Maintenance Setup

Architecture Strategy: Hybrid (Separated Engine)

Verification: Why This Layout Wins

Forcing an entire live TV playlist into Kodi creates a massive performance lag. Keeping TiviMate completely separate guarantees your live TV operates smoothly every day. The Onn 4K Pro's upgraded 3GB of RAM provides more than enough headroom to toggle instantly between VOD and Live TV using the remote control shortcut keys.

Phase 1: Hardware & External Storage Expansion (DVR Setup)

The Onn 4K Pro features a native USB 3.0 Port. To format a high-capacity USB Flash Drive or External Hard Drive to work smoothly as a storage location for TiviMate DVR records, complete the following steps:

Step 1: Prep the System & Mount

  • Insert your external drive or USB stick directly into the back port of your Onn 4K Pro.
  • Go to Settings > System > Developer options. Ensure that USB Debugging is turned OFF. If left on, the system will not read the external drive files correctly.

Step 2: Storage Configuration Choice

Option A: Removable Storage (Recommended) Keep the drive formatted as FAT32 or exFAT. This keeps your files readable by computers and makes backups easy.
Option B: Adopted Device Storage Go to Storage settings and choose "Format as device storage". This merges the drive space completely into the internal system memory.
Crucial Android TV 12 Scoped Storage Bypass: Android TV heavily limits write access to external drives. To ensure TiviMate actually saves your recorded files, install the X-plore File Manager app from the Play Store. Use it to navigate to your external storage directory and create a brand-new folder named exactly TiviMate (capitalized exactly as shown) inside the root directory.

Phase 2: Operating System Optimization

Strip out background lag and clear corporate ads off the device dashboard before loading your software setup:

  1. Unlock Developer Access: Navigate to Settings > System > About. Scroll down to Android TV OS Build and hit the center selection button 7 times consecutively.
  2. Enable Unknown Sources: Go to Settings > Apps > Security & Restrictions > Unknown Sources and grant toggle access to your download utilities.
  3. Enable Apps-Only Mode: Navigate to Settings > Accounts & Sign-In > Select your profile ID > Toggle Apps-only mode to ON. This clears out heavy video trailers and menu recommendations.
  4. 0.5x UI Speed Optimization: Enter your new Developer options menu. Locate Window animation scale, Transition animation scale, and Animator duration scale. Drop each setting from 1.0x down to 0.5x.

Phase 3: The Video-on-Demand (VOD) Engine Setup

The Brain: Trakt.tv Synchronization (Mandatory)

Trakt is required to sync watch history, build the "Up Next" episode lists, and populate the widgets on the home screen. Chris does NOT need to give David his password. Use the API PIN method below during setup.

Choose between Kodi (with the lightweight Nimbus Skin) or Stremio to handle your movies and TV show libraries:

Option 1: Kodi 21 (Omega) + Nimbus Skin + Trakt

  1. Install Kodi 21 Omega from the Google Play Store.
  2. Go to Settings (Gear Icon) > File Manager > Add Source.
  3. Select <None> and type: https://ivarbrandt.github.io/repository.ivarbrandt/
  4. Name it ivarbrandt and click OK.
  5. Go to Add-ons > Install from Zip File > ivarbrandt and install the zip.
  6. Go to Install from Repository > ivarbrandt repository > Look and Feel > Skin > Select Nimbus and install. Accept the prompt.
  7. Link TorBox: Open your video add-on (Fen Light/Umbrella) > Settings > Accounts > Enter your TorBox API Key.
  8. Link Trakt: In the add-on settings > Accounts > Click "Authorize Trakt". A popup with an 8-digit code will appear on the TV. Have Chris go to trakt.tv/activate on his phone and type in the code.
  9. Configure Nimbus widgets: Skin Configuration > Main Menu Items > Point paths to your Trakt add-on lists.

Option 2: Stremio Setup (Lighter Alternative)

  1. Install the official Stremio client app directly from the Google Play Store.
  2. Create an account and sign in.
  3. Link Trakt: Go to Stremio Settings (Gear Icon) > Scroll down to Trakt > Click Authenticate. Chris can approve this via his phone.
  4. On a phone/computer browser, go to the Stremio Add-ons store and configure Torrentio or AIOStreams.
  5. During add-on setup, select TorBox as the Debrid Provider and paste your API Key. Install it.

Phase 4: Live TV (TiviMate) Integration & Remote Mapping

Finalize the build by connecting your live streaming content and linking your physical hardware keys:

  1. Install TiviMate from the official storefront and activate your premium subscription.
  2. Add your custom playlist file string (M3U URL or Xtream Codes credentials).
  3. Map the DVR Recording Directory: In TiviMate, navigate to Settings > Other > Recording > Recording Folder > Select Folder. Navigate to your external storage directory and select the dedicated TiviMate folder you built in Phase 1.
  4. Configure the Magic Remote Shortcut: Go out to the main device system settings: Settings > Remotes & Accessories > Setup Magic Button. Link the physical Star (⭐) button on your Onn remote to instantly launch your VOD app (Kodi or Stremio).
