Onn 4K Pro Master Deployment Blueprint

The Blueprint for David & Chris's Low-Maintenance Streaming Setup

Architecture Strategy: Hybrid (Separated Engine)

Verification: Why This Layout Wins

Forcing an entire live TV playlist into Kodi creates a massive performance lag. Keeping TiviMate completely separate guarantees your live TV operates smoothly every day. The Onn 4K Pro's upgraded 3GB of RAM provides more than enough headroom to toggle instantly between apps using the remote control shortcut keys.

Phase 1: Hardware & External Storage Expansion (DVR Setup)

The Onn 4K Pro features a native USB 3.0 Port. To format a high-capacity USB Flash Drive or External Hard Drive to work smoothly as a storage location for TiviMate DVR records without triggering Android security errors, complete the following steps:

Step 1: Prep the System & Mount

  1. Insert your external drive or USB stick directly into the back port of your Onn 4K Pro.
  2. Go to Settings > System > Developer options. Ensure that USB Debugging is turned OFF. If this option is left on, the system will not read the external drive files correctly.

Step 2: Storage Configuration Choice

You have two setup choices depending on how you intend to use the extra space:

Option A: Removable Storage (Recommended)
Keep the drive formatted as FAT32 or exFAT. This keeps your files readable by computers and makes backups easy.
Option B: Adopted Device Storage
Go to Settings > System > Storage, select your drive, and choose "Format as device storage". This merges the drive space completely into the internal system memory.
Crucial Android TV 12 Scoped Storage Bypass:
Android TV heavily limits write access to external drives. To ensure TiviMate actually saves your recorded files, install the X-plore File Manager app from the Google Play Store. Use it to navigate to your external storage directory and create a brand-new folder named exactly TiviMate (capitalized exactly as shown) inside the root directory or inside the Movies subfolder.

Phase 2: Operating System Optimization

Strip out background lag and clear corporate ads off the device dashboard before loading your software setup:

  1. Unlock Developer Access: Navigate to Settings > System > About. Scroll down to Android TV OS Build and hit the center selection button 7 times consecutively until it verifies your developer access.
  2. Enable Unknown Sources: Go to Settings > Apps > Security & Restrictions > Unknown Sources and grant toggle access to your download utilities.
  3. Enable Apps-Only Mode: Navigate to Settings > Accounts & Sign-In > Select your profile ID > Toggle Apps-only mode to ON. This clears out heavy video trailers and menu recommendations.
  4. 0.5x UI Speed Optimization: Enter your new Developer options menu. Locate the three animation configuration fields: Window animation scale, Transition animation scale, and Animator duration scale. Drop each setting value from 1.0x down to 0.5x.

Phase 3: The Video-on-Demand (VOD) Engine Setup

Choose between Kodi (with the lightweight Nimbus Skin) or Stremio to handle your movies and TV show libraries:

Option 1: Kodi 21 (Omega) + Nimbus Skin Installation

  1. Install Kodi 21 Omega from the Google Play Store or sideload it using Downloader.
  2. Launch Kodi, go to Settings (Gear Icon) > File Manager > Add Source.
  3. Select <None> and type this URL precisely: https://ivarbrandt.github.io/repository.ivarbrandt/
  4. Name the custom source path ivarbrandt and click OK.
  5. Return to the main settings page, select Add-ons > Install from Zip File. Select ivarbrandt and execute the repository zip file installation.
  6. Select Install from Repository > Open the ivarbrandt repository > Navigate to Look and Feel > Skin > Select Nimbus and click install. Accept the prompt to swap skins.
  7. Open the video add-on of your choice (e.g., Fen Light or Umbrella) inside Kodi, open its settings menu, select Accounts, and log in using your TorBox or Real-Debrid API Key.
  8. Configure your Nimbus home page widgets by going to Skin Configuration > Main Menu Items and setting your display paths to point to your add-on lists.

Option 2: Stremio Setup (Lighter Alternative)

  1. Install the official Stremio client app directly from the Google Play Store.
  2. Create an account and sign in.
  3. On a phone or computer browser, go to your Stremio Add-ons store and configure the Torrentio or AIOStreams add-on.
  4. During the add-on setup, select TorBox or Real-Debrid from the Debrid Provider dropdown field and paste your unique API Key. Click Install. Your entire library layout will sync automatically to your TV.

Phase 4: Live TV (TiviMate) Integration & Remote Mapping

Finalize the build by connecting your live streaming content and linking your physical hardware keys:

  1. Install TiviMate from the official storefront and activate your premium subscription access.
  2. Add your custom playlist file string (M3U URL or Xtream Codes login credentials).
  3. Map the DVR Recording Directory: In TiviMate, navigate to Settings > Other > Recording > Recording Folder > Select Select Folder. Navigate to your external storage directory and select the dedicated TiviMate folder you built in Phase 1.
  4. Configure the Magic Remote Shortcut: Go out to the main device system settings: Settings > Remotes & Accessories > Setup Magic Button. Link the physical Star (⭐) button on your Onn remote to instantly launch your preferred VOD client app (Kodi or Stremio) from anywhere on the device.
